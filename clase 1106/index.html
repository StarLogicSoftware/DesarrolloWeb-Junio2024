Párrafo de texto

Otros temas que vimos en la primer clase

Extensiones que vamos a instalar el el editor VSCode

"Live Server" para crear un servidor web local para nuestro sitio

"Spanish languaje" para traducir el editor al español

Guardar el archivo en el que estamos trabajando

Lo hacemos con el atajo de teclado "Ctrl + S" o con el menú "Archivo -> Guardar"

Formatear el texto en el editor cuando es muy largo y no cabe en pantalla

Si hacemos, por ejemplo un parrafo con un lorem200, vamos a ver que el texto se "escapa" del editor, lo podemos arreglar presionando "Alt+Z" para activar o desactivar, o el menú "Ver -> Ajuste de palabra"

Para agregar texto de prueba usamos "LoremX"

Donde el XXX lo reemplazamos por el numero de palabras que queremos, por ejemplo "Lorem2" agrega un lorem de 2 palabras, y un "Lorem20" lo hace con 20 palabras útil para textos de relleno

Ejemplo lorem2

Lorem, ipsum.

Ejemplo lorem20

Lorem ipsum dolor sit amet consectetur adipisicing elit. Odio, totam sed earum pariatur laudantium laboriosam architecto molestias laborum beatae sequi.

Nombre del archivo principal

Si hacemos una pagina, debemos llamarla "index.html", index es para que el servidor la tenga como página por defecto (si o si debemos tener una) y la extensión ".html" es para indicar que usamos el lenguaje HTML para crear la página

Ejecutar la pagina para verla en el navegador
